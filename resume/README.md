# Резюме (React + Vite)

## Тексты и языки

- **EN:** [`src/i18n/locales/en.ts`](src/i18n/locales/en.ts)
- **RU:** [`src/i18n/locales/ru.ts`](src/i18n/locales/ru.ts)

Переключатель **EN | RU** в шапке; выбор хранится в `localStorage` (`resume-locale`). По умолчанию: `ru`, если язык браузера начинается с `ru`, иначе `en`.

Типы и константы: [`src/i18n/types.ts`](src/i18n/types.ts), [`src/i18n/constants.ts`](src/i18n/constants.ts).

## Команды

```bash
npm install
npm run dev
npm run build
```
